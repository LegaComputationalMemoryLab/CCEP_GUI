# CCEP_GUI
Python-based CCEP analyzer for presurgical epilepsy mapping. Imports EDF recordings and SPES logs, detects pulses, preprocesses iEEG, computes ERP/RMS/low-gamma metrics, links responses to ROI-based 3D anatomy, and exports static figures, interactive HTML reports, timing logs, and auditable channel-level summaries. Designed for clinical review use.
