# License options to confirm before public release

Confirm license choice with the principal investigator and institutional legal/technology-transfer office before making the repository public.

## Common options

### MIT License

Simple permissive license. Allows reuse, modification, distribution, and sublicensing with attribution and warranty disclaimer.

### BSD 3-Clause License

Permissive academic license. Similar to MIT but includes a non-endorsement clause.

### Apache License 2.0

Permissive license with explicit patent grant. More detailed than MIT/BSD.

## Not recommended without a specific reason

- No license: others have no clear right to reuse the code.
- GPL/copyleft licenses: may be inappropriate if the lab wants permissive clinical/research reuse.
- Creative Commons licenses for software: generally better for text/media than executable code.
